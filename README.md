## Resumo do projeto

Leito de PDF simples, ele pega todos os conjuntos numericos de 5 numeros, fiz para pegar todos os codigos da um catalago de onde eu trabalho atualmente.

## Tecnologias

- `Intellij`
- `Java 21`
- `Maven`
- `PDFBox`
